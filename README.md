# 文字数カウンター
- ![test]
- このプログラムは空白を無視して入力したテキストの文字数をカウントします。

- 使用方法
  - 1 countに実行権限を付与する。 $ chmod +x count.py を実行
  - 2 $ ./count.py　でプログラムを実行する。
  - 3 文字数を知りたいテキストを入力し、Enterキーを押すと文字数が表示される。
  - 4 Ctrl + c で終了する。
